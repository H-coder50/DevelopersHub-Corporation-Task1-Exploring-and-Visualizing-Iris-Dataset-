Multiple plots were created to understand the dataset:
-	Scatter Plot:
 To observe relationships (e.g., sepal length vs sepal width) and species clustering.
-	Histogram:
 To view distribution patterns of individual numerical features.
-	Box Plot:
 To detect outliers and compare feature spreads.
-	Pairplot:
 To visualize all feature relationships with species color separation.

### Results & Insights:

From the visualizations, several insights were discovered:
- 1. Species Separation
  -	The Setosa class is clearly separable from the other species.
  -	Versicolor and Virginica show partial overlap but still have distinguishable patterns.
  -	Petal features (length & width) separate species much better than sepal features.
- 2. Distribution Characteristics
  -	Petal length has a strong variation and distinct peaks for each species.
  -	Sepal features show overlapping ranges across species.
- 3. Outliers
  -	Box plots revealed minimal outliers, indicating the dataset is clean and suitable for ML tasks.
- 4. Relationship Trends
  -	Scatter plots show linear relationships among some features.
  -	Pairplots clearly display three clusters corresponding to species.
